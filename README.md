#Responsive Landing Page - Next.js

technical demonstration of a responsive landing page.

#Tech Stack
- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Icons: Phosphor Icons

#Key Features
- Mobile-First Design: Fully responsive layout for all screen sizes.
- Semantic HTML: Proper use of `<nav>`, `<main>`, and `<section>` tags.
- Modern UI: Clean code with hover transitions and optimized typography.
